# CRIME-TREND-FORECASTING-AND-CLASSIFICATION
# 📊 Crime Trend Forecasting and Classification in Kerala

## 📌 Overview

This project focuses on **forecasting and classifying crime trends in Kerala** using machine learning techniques. It uses historical crime data (2016–2025) to predict future trends (2026–2028) and classify crime severity into **high** and **low** categories.

The system helps in understanding crime patterns and supports **data-driven decision-making** for law enforcement and policymakers.

---

## 🚀 Features

* 📈 Crime trend forecasting (2016–2028)
* 🤖 Machine Learning models:

  * Linear Regression
  * Random Forest
  * XGBoost
* ⚖️ Binary classification (High vs Low crime years)
* 🔄 SMOTE for handling imbalanced datasets
* 📊 Data visualization:

  * Heatmaps
  * Trend plots
  * Confusion matrices
* 🧠 Model evaluation using Accuracy, Precision, Recall, F1-score, ROC-AUC

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/crime-forecasting.git
cd crime-forecasting
```

### 2. Create a virtual environment

```bash
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```
